# 🧪 Laboratorio Lenguaje C

## 📌 Descripción

Este proyecto corresponde al laboratorio de **Fundamentos de Programación en C**, cuyo objetivo es aplicar conceptos clave del lenguaje mediante el desarrollo de una librería y una estructura de datos dinámica.

El laboratorio se divide en dos partes:

- 🔹 Primera parte: Implementación de una librería general (`global.h`)
- 🔹 Segunda parte: Implementación de una lista enlazada de estudiantes

---

## 📂 Estructura del Proyecto

La entrega debe respetar estrictamente la siguiente estructura:
entrega_<grupo>/
├── primera_parte/
│ ├── global.h
│ ├── global.c
│ └── main.c
└── segunda_parte/
├── estudiantes.h
├── estudiantes.c
└── main.c
